Click anywhere on the field to place beacons, then click start to start the simulation.
It is wise to place the beacons in a circle-like shape.
Drain speed (to change the speed at which batteries drain)
Robot speed (to change the speed at which the robot travels)
Robot charge (to change the speed at which the robot charges)
Warning level (to change the battery level when a beacon cries for help)
  • BN: Beacon Number
  • CH: Battery Charge
  • QL: Battery Quality
  • SS: Signal Sent
  • SR: Signal Received
  • RW: Robot is on the Way
  • RH: Robot is Here
  • RC: Robot is Charging the beacon
